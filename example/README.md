# Omikuji (おみくじ) - Japanese Fortune Telling App

A simple web-based Japanese fortune telling application.

## Features

- Click the omikuji box to draw your fortune
- Six fortune levels: 大吉, 吉, 中吉, 小吉, 末吉, 凶
- Beautiful animated UI with gradient backgrounds
- English and Japanese text support
- Fully responsive design

## How to Use

Simply open `index.html` in your web browser and click on the red omikuji box to draw your fortune!

## Fortune Meanings

- **大吉**: Excellent fortune - Best luck!
- **吉**: Good fortune - Things will go well
- **中吉**: Moderate fortune - Patience will be rewarded
- **小吉**: Small fortune - Appreciate small joys
- **末吉**: Future fortune - Good things coming later
- **凶**: Bad fortune - Be cautious

## Technical Details

- Pure HTML, CSS, and JavaScript
- No external dependencies required
- Works in any modern web browser
