# おみくじアプリ

シンプルなウェブ版おみくじアプリケーションです。

## 機能

- 6種類の運勢（大吉、吉、中吉、小吉、末吉、凶）
- インタラクティブなシェイクアニメーション
- レスポンシブデザイン（デスクトップ・モバイル対応）
- ラッキーカラー・ラッキーナンバー表示

## 使い方

1. `example/index.html` をブラウザで開きます
2. おみくじ箱をクリックして運勢を引きます
3. 結果を確認し、「もう一度引く」ボタンで再チャレンジできます

## 技術仕様

- 純粋な HTML/CSS/JavaScript で実装
- 外部ライブラリ不要
- モダンブラウザで動作
